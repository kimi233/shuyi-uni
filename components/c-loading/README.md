# c-loading
作者：kimi233

Email: 15870290085@163.com

一个统一的 loading 组件，loading动画基于 `c-loading-animation` 组件

> 该组件已封装到 c-page 组件中，推荐直接调用  c-page 提供的方法。具体文档请到 c-page 组件查看

## 使用方式

**在 template 中使用组件**

```vue
<c-loading></c-loading>
```

# 属性说明



## props

|属性名 |是否必填 |	类型 |	默认值 |	说明 |
|-------|---------|--------|-----|-----|
|title |否 |	String |	请稍后 |	loading中的文字信息 |
|show |否 |	Boolean |	false |	是否显示loading |
