# c-navbar

作者：kimi233

Email: 15870290085@163.com

自定义导航栏组件，由 `u-navbar` 组件源码改装而产生。相对与 `u-navbar`组件，`c-navba` 组件对页面返回逻辑做了改装（原来的页面返回逻辑会让 `c-page` 组件的一部分全局组件调用方法失效。）

## 使用方式

**在 template 中使用组件**

```vue
<c-navbar></c-navbar>
```
