作者：月夕

这里是 c-tabbar 组件的基本作用,记得写这个组件是干嘛的。

## 使用方式
**在 script 中引用组件**

```vue
import Ctabbar from "@/components/c-tabbar"
export default {
    components: { Ctabbar }
}
```

**在 template 中使用组件**

```vue
<c-tabbar></c-tabbar>
```
