# c-loading-animation

作者：kimi233

Email: 15870290085@163.com

一个 loading 动画，全局应统一使用该动画作为 loading 动画

## 使用方式

**在 template 中使用组件**

```vue
<c-loading-animation></c-loading-animation>
```

|属性名 |是否必填 |	类型 |	默认值 |	说明 |
|-------|---------|--------|-----|-----|
|user-style |否 |	String |	 |	通过width和height控制大小 |
